## 📦 如何使用 (小白防坑配置指南)

### 第一步：克隆源码与安装依赖
请确保你的电脑上安装有 `Node.js >= 18` 环境。

\`\`\`bash
git clone https://github.com/code-wgl/pdgzf-renter-map.git
cd pdgzf-renter-map
npm install
\`\`\`

### 第二步：申请并替换自己的【高德地图 API Key】
本大屏强依赖**高德地图**进行房源打点、渲染与实时防拥堵测距。默认 Key 若遭大流量访问可能会受限或封锁，强烈建议在本地运行时换成您专用的免费 Key：
1. 访问 [高德开放平台控制台](https://console.amap.com/dev/key/app) 登入。
2. 进入“应用管理 -> 我的应用”，点击“创建新应用”，随后为应用添加一个 Key。
3. ⚠️ **服务平台一定要选择 【Web端 (JS API)】**。
4. 将生成的一串 Key 号码复制下来。
5. 打开本项目源码区中的 `src/App.jsx` 文件（大约在第 32 行），替换即可：
   \`\`\`javascript
   const amapConfig = {
     key: "替换为您自己新申请的这串高德 Key", 
     version: "2.0",
     plugins: ['AMap.MarkerCluster', 'AMap.Driving']
   };
   \`\`\`

### 第三步：如何更换为您自己的主副中心（公司坐标）？
打开 `src/App.jsx` 的文件头部，把默认的传音控股改掉！
*(您可以在百度/高德【坐标拾取器】网站上获取您想定位的经纬度)*
\`\`\`javascript
// 修改成您公司自己的坐标 [经度, 纬度]
const COMPANY_LOCATION = [121.613589, 31.185108]; 
const COMPANY_NAME = "我的公司大楼"; 
\`\`\`

## 🛠️ 数据捕获方案 (获取房源的最速秘籍)
这套系统的核心难点是如何将【需要登录状态】的公租房官方数据优雅地喂进这块板子里。本项目为您构筑了两大流派玩法：

### 方案 A：极客懒人的终极救赎 —— 【一键全息爬虫】 (超强力推荐)
我们为您定制了一个内置的无差别自动化爬虫环境：
1. 确保在系统终端打下：
   \`\`\`bash
   npm run fetch
   \`\`\`
2. 此时，系统像变魔术般自动唤出一个 Chromium 浏览器并停留在公租房账户登录页面。
3. 您仅需在这个不受干扰的官方网页上，安全地**扫码/手工完成登录**，并像平时一样点击“全区房源地图”。
4. **无需 F12，无需 Curl！** 这个浏览器底层埋设的监听器会在数据流下来的那一刹那，拦截总计高达上千套房源的庞大 JSON 包！强制写入并覆盖到本系统的 `src/output.json`，完毕后安静自毁关闭进程。
5. 数据拿到了，立刻启动前端大屏：
   \`\`\`bash
   npm run dev
   \`\`\`

### 方案 B：硬核黑客/云端部署客 —— 【纯净前端沙箱拖拽流】
若是您想把大屏做成了 **GitHub Pages** 或 **Vercel** 上的无后端纯前端静态死页面（Serverless），或者极度反感运行爬虫脚本，那就利用此原教旨玩法：
1. 打开浏览器正常访问 `https://select.pdgzf.com/`，按 `F12` 唤出检查器。
2. 正常查询房源，在 Network 板块找到名称为 `GetStatistics` 的接口。
3. `右击 -> Copy as cURL`，然后你可以随意使用 Postman、Apifox 或者您本地写个脚本拿回庞大的 JSON，甚至直接在浏览器的 Response 栏右击复制存为一个名为 `任意名字.json` 的文本文件。
4. 点开您的大屏网页（随便在哪里托管），点击右上角的 **【📂 导入新数据】**！
5. 选中您的文件喂给它！大屏会在浏览器内瞬间将这几兆文本拆解，调用 LocalStorage 全包接盘！
   *👉 完全不用网络开销发送数据到后台，零服务器算力白嫖渲染，全站长久免疫重新配置。真正实现了无端数据更新的最短路径！*
